# 🔐 Caesar Cipher Tool

A complete Caesar Cipher encryption/decryption tool with both **Web Interface** and **Terminal Version**.

## 🎯 Features

- ✅ **Encrypt** messages with custom shift value (1-25)
- ✅ **Decrypt** messages using the correct shift value
- ✅ **Brute Force** - Try all 25 possible shifts automatically
- ✅ **Beautiful Web UI** - Gradient background, responsive design
- ✅ **Terminal Version** - For command-line users
- ✅ **Preserves** numbers, spaces, and special characters

## 🚀 Live Demo

Open `index.html` in any web browser to use the tool.


## 💻 How to Use

### Web Version (Recommended)
1. Double-click `index.html`
2. Type your message
3. Click **Encrypt**, **Decrypt**, or **Brute Force**
4. View result instantly

### Terminal Version
```bash
python caesar_cipher.py
